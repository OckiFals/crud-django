# crud-django
Django CRUD sample applications

### Required Django >= 1.9.2

Instalation
---------------
1. Unarchive into one folder
2. Locate that folder
3. Fix dependencies

`pip install -r requirements.txt`

Run Program
---------------
Windo** 

`python manage.py runserver`

Linux

1. `chmod +x manage.py`
2. `./manage.py runserver`
